Welcome to NOME!

What is this place?

NOME is a site created by Caden Henrich to host programming ideas and projects. The site includes projects in JavaScript, Python, and C++. This site's purpose is to prove that programming isn't just fun, it can be artistic too!

What is flask?

Flask is a lightweight framework for building and hosting websites in Python. This page uses flask to host and handle requests, and HTML5/CSS3 to build and style the site. To learn more about flask, visit the official flask site.
